-----------------------------------------------------------------------

## DESCRIPTION OF PROJECT:
Implementation of Kruskals and prims, 2 algorithms.
-----------------------------------------------------------------------

## INPUT:
Wieghted graph adjacency matrix is created with random graph size and random edge wieghts.
-----------------------------------------------------------------------

## OUTPUT:
A minimum spanning tree and the nodes and edges between these nodes in the minimum spanning tree.
-----------------------------------------------------------------------
## FILES INCLUDED:
	1. Makefile:
		makefile to compile the program.
	2. singh_himanshu_proj4.c:
		Implementation of kruskals and prims algorithms. Wieghted graph adjacency matrix 
		is created with random graph size and random edge wieghts and kruskals and prims algorithm is demonstrated on this graph, creates a minimum spanning tree and output is the nodes and edges between these node in the minimum spanning tree.
-----------------------------------------------------------------------

## COMPILING THE CODE:
fire the make command the executable with name singh_himanshu_proj4 will be produced.
-----------------------------------------------------------------------

## RUNNING THE CODE:
launch the executable singh_himanshu_proj4.
