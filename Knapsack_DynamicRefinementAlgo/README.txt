-----------------------------------------------------------------------

## DESCRIPTION OF PROJECT:
Implmentation of 0/1 knapsack problem using three methods: brute force, dynamic programming, backtracking algorithm.
-----------------------------------------------------------------------

## INPUT:
The code generates a bag with random number of bag items and each having random wieght and value.
The generated backpack is printed on screen.
-----------------------------------------------------------------------
## OUTPUT: 
The output of the code is 
	1. Brute force: 
		items included in backpack 
		total wieght and value of backpack.
	2. Dynamic Programming with refinement: 
		items included in backpack  
		value of backpack 
		table of the values calulated by algorithm
	3. Bcktracking:
		items included in backpack 
		value of backpack
-----------------------------------------------------------------------

## FILES INCLUDED:
1. Makefile: for compiling the code.

2. singh_himanshu_proj5.c: contains the implementation of following algorithms to solve the 0/1 knapsack problem:
	1. Brute force
	2. Refinement Dynamic programming (Refinement: only the relevant values in table are calculated).
	3. Backtracking algorithm.
-----------------------------------------------------------------------			

## COMPILING THE CODE:
fire the make command the executable with name singh_himanshu_proj5 will be produced.
